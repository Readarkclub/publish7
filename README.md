
  # 活动发布平台首页设计

  This is a code bundle for 活动发布平台首页设计. The original project is available at https://www.figma.com/design/CSsDUydx3Db96fNwi7Eyww/%E6%B4%BB%E5%8A%A8%E5%8F%91%E5%B8%83%E5%B9%B3%E5%8F%B0%E9%A6%96%E9%A1%B5%E8%AE%BE%E8%AE%A1.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deployment

  This project is configured to deploy on Vercel. Environment variables are required for Supabase integration.
